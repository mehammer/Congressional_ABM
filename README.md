# Congressional_ABM
Github repository for Michael Hammer's dissertation on simulating U.S. Congressional Elections, defended on November 14, 2025 at George Mason University for the degree of Ph.D. in Computational Social Science.

Instructions for running each model are in the chapter folder in the overview.txt file.

Chapter 2: Mdoel and instructions are available for the "base model" discussed in Chapter 2.

Chapter 3: Jupyter notebooks, files, and instructions for the text analysis machine learning models discussed in Chapter 3. Instructions for acquiring training dataset can be found in the README.pdf file in the Chapter 3 folder. 

Chapter 4: Models and instructions are available for the ten congressional districts modeled in Chapter 4.

Chapter 5: Models and instructions are available for the "top two" reform simulations for ten congressional districts modeled in Chapter 5.
